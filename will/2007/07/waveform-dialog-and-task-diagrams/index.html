Waveform Dialog and Task Diagrams

by @im_a_muppet on July 24, 2007

The Dialog and Task diagrams are displayed below. While developing the TAC Palette and then these diagrams from the conceptual model I was able to get a better understanding of the users interaction with the system. Thus TUIML helps the development of the project by defining exactly what the system should be able to do or not do. The only problem I have with the TUIML is that I am not sure how to express tangible outputs (e.g. the spinner and the light).
